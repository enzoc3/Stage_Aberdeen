    What is a pixel?

    A pixel is defined by three integer values between 0 and 255 which represents the red, green and blue values of the pixel. For example, a white pixel is defined by (255, 255, 255), a black pixel by (0, 0, 0), a classic red pixel by (255, 0, 0), a classic green pixel by (0, 255, 0) and a classic blue pixel by (0, 0, 255).

    Another value, less known and used is the alpha value of the pixel also defined between 0 and 255. It is the degree of transparency of the pixel. The value 0 is equal to a transparency of 100%, so a transparent pixel. The value 255 is a pixel totally visible, so a transparency of 0%. The alpha can be used for example to modify the light of the pixel.

    A pixel alone can’t be called a picture. A picture is composed of many pixels.

    Example of a red pixel 50% transparent: (255, 0, 0, 127)

    Example of a black pixel 50% transparent: (0, 0, 0, 127)

    What is a picture?

    A picture is defined with its width and height values both as integer. There are the number of pixel in the width and in the height in the pixel. In fact, a picture is an array of pixel with width*height pixel in total. A picture is write and read to the top corner left to the bottom corner right, as a book.

    Example of a white picture of 3x3:
    (255, 255, 255) (255, 255, 255) (255, 255, 255)
    (255, 255, 255) (255, 255, 255) (255, 255, 255)
    (255, 255, 255) (255, 255, 255) (255, 255, 255)

    Example of a black picture of 3x3:

    Property and functions of the class Pixel in our JavaScript

    As said, a pixel need some values: red, green, blue, alpha and its position x and y in the picture. The constructor of the class Pixel need the same number and order of properties. Of course, it isn’t possible to take a values smaller than 0 and higher than 255 for a colour. It is possible to reach all of them, with the call of property (“this.”) or with a function.

    To create a pixel black at coordinates 50, 50: var myPixel = Pixel(0, 0, 0, 255, 50, 50);
    To display some information about a pixel: myPixel.toString();


    To reach the value red with property: myPixel.this.red;
    To reach the value green with property: myPixel.this.green;
    To reach the value blue with property: myPixel.this.blue;
    To reach the value alpha with property: myPixel.this.alpha;
    To reach the position x with property: myPixel.this.x;
    To reach the position y with property: myPixel.this.y;
    To reach the colour with function: myPixel.getColor();
    To reach the position with function: myPixel.getPos();


    All of this properties can be modify, just with the call of this property or with a function which we have created.

    To modify the red value of the pixel with property: myPixel.this.red = 255;
    To modify the alpha value of the pixel with property: myPixel.this.alpha = 175;
    To modify all colours of the pixel to white with function: myPixel.setColor(0, 0, 0, 255);
    To modify all coordinates of the pixel with function: myPixel.setPos(100, 100);


    Property and functions of the class Picture in our JavaScript

    Before to draw or modify a picture, the picture must be existed. Create a picture needs an image in the same folder source as parameter. The format of the image can be a “png” or “jpg”. The path source of the picture can be modify. To use the picture, she is drawing in a canvas, a space in the web page. The size of this canvas was initialised to the size of the picture, but it is possible to modify it.


    To create a picture: var myPicture = Picture(“picture.jpg”);
    To display the first time a picture: myPicture.display();
    To draw the others time the picture: myPicture.redraw();
    To display some information about the picture: myPicture.toString();


    To reach the width with property: myPicture.this.width;
    To reach the height with property: myPicture.this.height;
    To reach the canvas with property: myPicture.this.canvas;
    To reach the path source with property: myPicture.this.source;
    To reach the colour of a pixel with function: myPicture.getPixel(50, 50);


    All of this properties can’t be modify, only the following can be changed, just with the call of this property or with a function which we have created. To modify the canvas, the function need the name of a canvas which existed in the page HTML. It isn’t possible to modify the size of the picture but is possible to change the size of the canvas and after modify pixels of the picture.

    What is a sound?


    A sound is represented by an array of float: the buffer. If you want to modify a sound you will be working on this array. The length of this array is determined by the duration and the sample rate of the sound.

    Properties and functions of the class Sound:


    The sound class possess various properties such as:
    • • audioCtxt : the audio context used to manage the sound
    • • source : the audio source used to play the sound
    • • soundBuffer : the audio data
    • • nbChannels : the sound’s number of channels, each channel is an array of float
    • • duration : the sound’s duration in seconds
    • • sampleRate : the sound’s sample rate
    • • length : the soundBuffer’s length
    To create a sound you will use the constructor: var mySound = new Sound(); Then if you want to play it there are 3 possibilities:
    • • mySound.play(loop) : this function play the sound once by default and play the sound in loop if the function is called with loop = true. The Boolean loop is a facultative argument
    • • mySound.startAt(time) : this function start the sound at the specified time (in seconds)
    • • mySound.generatePlayer() : this function generate a player composed of 4 button: play, stop, pause, loop.
    Now that you know how to create a sound and play it you may want to modify it. To do that you have 2 option: In the following line channelNb represent the number of the channel you want to modify, sampleIndex is the index in the channel that you want to obtain or modify.
    • • The first option is to use these functions: getSample(channelNb,sampleIndex) and setSample(channelNb,sampleIndex,value). getSample(channelNb,sampleIndex): returns the value at sampleIndex of channelNb (be careful the first channel is 0) setSample(channelNb,sampleIndex,value): replaces the value at sampleIndex of channelNb (be careful the first channel is 0) with value
    • • The second option is to change the whole channel using getChannel(channelNb) and setChannel(channelNb, array)
